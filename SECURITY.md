# Security Policy

## Reporting a vulnerability

If you discover a security issue, please report it privately rather than opening a public issue.

Use GitHub private vulnerability reporting for this repository:

- https://github.com/lachieJ/APartner/security/policy

Please include:

- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any suggested remediation.

## Disclosure guidance

- Do not publicly disclose vulnerabilities until maintainers confirm a fix is available.
- Provide reasonable time for triage and remediation.
