# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and empathetic.
- Give and receive feedback constructively.
- Focus on what is best for the community and project.

## Unacceptable behavior

- Harassment, discrimination, personal attacks, or intimidation.
- Trolling, insulting comments, or disruptive conduct.
- Publishing others’ private information without explicit permission.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take appropriate action for violations.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions.
